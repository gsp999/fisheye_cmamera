# 鱼眼相机级联检测系统 (Fisheye Cascade Vision System)

本项目是一套针对鱼眼摄像头设计的视觉识别框架，专门解决鱼眼镜头边缘畸变严重导致 YOLO 模型（OBB/Pose）识别率下降的问题。通过“先去畸变/投影变换，后级联检测”的策略，实现高精度的关键点提取。

## 🌟 核心功能

* **多方案投影变换**：内置等距柱状 (Equirectangular)、透视 (Rectilinear) 和立方体 (Cubemap) 三种投影方式，灵活应对不同视野需求。
* **级联检测引擎 (`CascadeDetector`)**：
    1.  **第一级**：利用 YOLOv8-OBB 在投影图中定位旋转目标。
    2.  **第二级**：根据 OBB 结果提取目标 ROI，并进行仿射变换标准化（256x256）。
    3.  **第三级**：在标准化图中运行 YOLOv8-Pose 提取高精度关键点。
* **坐标双向反推**：自动完成从“抠图坐标 -> 投影图坐标 -> 原始鱼眼坐标”的全链路映射。
* **自动化标定**：提供配套脚本，一键生成鱼眼相机内参和畸变参数。

## 🛠️ 三种投影模式说明

在 `main.py` 的主程序入口处，你可以通过切换 `projector` 的实例化对象来选择模式：

1.  **方案 A: 等距柱状投影 (Equirectangular)**
    * **特点**：将 180° 视野展开为宽幅全景图。
    * **场景**：需要全局视野监控。
2.  **方案 B: 透视投影 (Rectilinear)**
    * **特点**：传统去畸变，线条拉直，符合人眼视觉。
    * **场景**：目标位于画面中心，且需要标准透视关系。
3.  **方案 C: 立方体投影 (Cubemap) [核心推荐]**
    * **特点**：将视野投影至立方体 6 个面并展开。
    * **场景**：**解决边缘畸变的终极方案**。能保持局部特征不失真，显著提升 YOLO 对画面边缘目标的识别率。

## 🚀 快速开始

### 1. 相机标定
首先需要获取你相机的物理参数：
1.  拍摄至少 15-20 张不同角度的棋盘格照片，放入 `/home/gsp/00fish_eye_camera/photo/`。
2.  运行标定脚本：
    ```bash
    python biaoding.py
    ```
    生成的 `fisheye_calib_params.npz` 会保存内参 $K$ 和畸变系数 $D$。

### 2. 运行实时检测
确保模型路径（`.pt` 文件）正确，然后启动主程序：
```bash
python main.py
